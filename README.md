# Mood Cycle Tracker 🧠

シンプルな双極性障害（躁鬱）向け周期管理・体調記録アプリです。
外部サーバーを使わず、ブラウザの `LocalStorage` にデータを保存するため、プライバシーを守りながら手軽に利用できます。
PWA（Progressive Web App）に対応しており、スマートフォンのホーム画面に追加してアプリのように使うことも可能です。

## ✨ Features

* **📅 カレンダー表示**: 月ごとの気分の波をヒートマップ風の色分けで視覚化。
* **📝 簡単記録**: 日付をタップして、気分（-5: 重い鬱 〜 +5: 激しい躁）とメモを記録。
* **🔒 プライバシー重視**: データは全てお使いの端末（ブラウザ）内に保存されます。外部への送信はありません。
* **💾 CSVエクスポート**: 記録したデータをCSV形式でダウンロード可能。主治医への相談資料作成などに役立ちます。
* **📱 PWA対応**: `manifest.json` と `sw.js` を配置することで、オフライン動作およびホーム画面への追加が可能です。

## 🚀 Quick Start

このリポジトリのコードは、特別なビルドツールを必要としません。

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くだけで動作します。
